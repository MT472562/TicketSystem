| プロパティ                     | 説明                                         |
|--------------------------------|----------------------------------------------|
| db_path                        | データベースのディレクトリとファイル名       |
| Ticket_Name                    | チケットの先頭文字列（例: 'TEST-TICKET1' など） |
| max_ppl                        | 登録できる最大人数                            |
| input_form_msg                 | 登録フォームに表示される文字列（空白可能）       |
| qr_file_path                   | 登録されたQRコードの保存ディレクトリパス      |
